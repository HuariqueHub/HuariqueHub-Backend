# PuntoSabor Backend

Backend desarrollado en ASP.NET Core para la gestión de huariques, usuarios, favoritos, reseñas, promociones, membresías y notificaciones.

## Tecnologías utilizadas

- ASP.NET Core .NET 8
- Entity Framework Core
- MySQL
- JWT Authentication
- Swagger

## Requisitos

- .NET SDK
- MySQL Server
- MySQL Workbench o phpMyAdmin
